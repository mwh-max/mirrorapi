Mirror API

About This Build

See what websites observe. Scroll speed, cursor trails, click pauses visualized in real time, explained with nuance.

INTERPRETATION

Mirror API shows how tracking scripts might interpret your everyday scrolling, hesitating or even moving your cursor. It doesn’t block them; it shows how they work.

DISTINCTION

Some privacy extensions block scripts. Mirror API lets you see yourself being seen.
